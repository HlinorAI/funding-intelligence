# Status verification

The handbook was checked on 20.07.2026. Statuses are routing hints, not live truth.

Before recommending `APPLY`:

1. Open the official program page.
2. Open the actual application/intake endpoint, not only a marketing page.
3. Confirm mechanism, eligibility, deadline/window and current terms.
4. Record the check date and link.
5. If official pages conflict, prefer `actual intake endpoint > current official announcement > docs > marketing page > aggregator`.

If the source is more than 7 days old, or a deadline/season is involved, mark the route `VERIFY` until checked again. Never infer an open program from a large treasury, old article, stale landing page or past funded cohort.
