# AI startup stage routing

| Stage | First routes | Evidence gap to close |
|---|---|---|
| `idea` | startup community, technical education, founder support | problem, team, prototype plan |
| `prototype` | cloud/API credits, technical programs, design-partner discovery | working demo, workload, first user/pilot |
| `early_product` | cloud credits, enterprise BD, targeted accelerator | repeat usage, customer proof, unit economics |
| `seed` | accelerator/investment, enterprise GTM, cloud-scale support | traction, growth, security, sales pipeline |
| `revenue` | enterprise procurement, strategic BD, investment | retention, revenue quality, expansion economics |

The table is a routing prior, not an automatic recommendation. Program-specific eligibility and anti-hallucination gates remain authoritative.
