# AI opportunity pack

Snapshot checked: 2026-07-22 against official program pages.

This is not an AI grant list. It distinguishes:

- startup and community programs;
- cloud or API credits;
- technical support and training;
- enterprise go-to-market access;
- accelerator and investment routes.

Credits reduce infrastructure burn; they are not runway or non-dilutive cash. An AI route must still pass stage, product, evidence, eligibility, and status gates.
